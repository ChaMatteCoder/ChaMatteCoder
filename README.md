## Projetos em destaque

### Cassino Entre Dimensões — Projeto Principal

Uma light novel interativa completa, construída como uma experiência narrativa rica em sistemas, estética e imersão.

O projeto apresenta uma saga dividida em episódios e capítulos, combinando mecânicas de magia, apostas, escolhas, progressão e ambientação cinematográfica. Além da narrativa, a aplicação conta com sistemas de álbuns, figurinhas, perfis, direção visual, organização de episódios e uma estrutura pensada para transformar leitura em experiência digital.

**Stack:** React, TypeScript, Node.js, Supabase, PostgreSQL
**Destaques:** narrativa episódica, sistema de álbuns e figurinhas, perfil de usuário, direção cinematográfica, universo autoral, experiência interativa
**Repositório:** [ChaMatteCoder/CassinoEntreDimensoes](https://github.com/ChaMatteCoder/CassinoEntreDimensoes)

### TeaToDo

Uma aplicação de produtividade local-first inspirada na calma do ritual do chá.

O TeaToDo reúne tarefas, listas, calendário, foco, hábitos e personalização em uma interface acolhedora, elegante e pensada para tornar a organização diária mais leve, fluida e consistente.

**Stack:** React, TypeScript, Local-first, CSS
**Destaques:** tarefas, listas, calendário, foco, hábitos, personalização, experiência acolhedora e organização local
**Repositório:** [ChaMatteCoder/TeaToDo](https://github.com/ChaMatteCoder/TeaToDo)

### ChAprendizagem

ChAprendizagem é um laboratório digital sobre aprendizagem de máquina.

O projeto combina explicações teóricas, visualizações interativas e pequenos experimentos de código para transformar conceitos de modelos em páginas navegáveis, acessíveis e fáceis de revisar.

**Stack:** JavaScript, CSS, Python, HTML
**Destaques:** machine learning, explicações teóricas, visualizações interativas, experimentos de código, páginas navegáveis para estudo
**Repositório:** [ChaMatteCoder/chaprendizagem](https://github.com/ChaMatteCoder/chaprendizagem)
